與社群聯繫

Contact with g0v Community

關於「與 g0v 合作」

台灣零時政府 ( g0v.tw ) 是一個開放的社群，不是公司、團體、組織、法人，主要由參與者自行以提案方式向其它社群參與者號召加入專案計畫。

因此，如果你有專案希望招募 g0v 參與者投入，可透過 g0v 社群的線上平台或黑客松活動以公開提案以及成果開源的方式進行。各種黑客松訊息請參考「活動訊息」頁面。

特定專案

此外，各個於 g0v 場域提案的專案 ( g0v 專案 ) 皆由專案發起人各自負責，若你對特定專案有興趣，可透過 g0v 社群的線上 / 線下平台尋找專案主 ( 見 聯繫管道 一節 )，洽談合作事宜。

以下例舉數個專案，以及其對應的聯繫窗口：

雙月大黑客松

社群目前最主要的活動「 g0v 雙月黑客松 」是由 g0v 揪松團 主辦。揪松團本身亦為一 g0v 專案，目標為引導社群發展並發揚 g0v 精神。

零時政府公民科技創新獎助金

約一年為一期、協助各公民科技專案發展的「零時政府公民科技創新獎助金」亦由 g0v 揪松團 執行。若有合作提案，歡迎與 揪松團 聯繫。

聯繫管道

若你無法直接聯絡上特定專案的發起人，亦可向社群求援；除了參與黑客松堵人外，也可透過線上社群平台詢問：

  • 臉書 g0v 後勤中心
  • 公共郵件論壇 g0v-general ( 較不活躍 )
  • g0v Slack 聊天室 ( 可透過 join.g0v.tw 取得 Slack 邀請郵件 )
  • IRC 線上聊天室
  • 或者，寫信給 g0v 揪松團 ( g0v-jothon[at]googlegroups.com ) 請求協助。

採訪/演講邀約

採訪或演講如無特定人選可來信至 g0v-talks 群組 ( g0v-talks [at] googlegroups.com ) ，社群將協調是否有適合人選。

g0v-talks 是？

g0v-talks 是一個任務性質的群組，工作內容為釐清演講與採訪邀約需求，轉發給適當的專案貢獻者。群組設定為非公開，目的是提供外界一個和社群私下聯繫的管道，處理牽涉到個人資料、個人隱私等不便在公開場合討論的事務。群組成立之初是為了處理演講邀約，後來隨著其他類型的邀約出現，演變為處理所有需要對外出席或發言的邀約，包括：演講、採訪、諮詢等。

想了解更多關於 g0v-talks 的事情，請參考 什麼是 g0v-talks 小組 / 零時政府媒體政策

注意事項

  • 黑客松不開放旁聽、採訪，想來現場請報名參與活動。
  • 社群採訪多採共筆形式，可參考過往媒體採訪共筆。
