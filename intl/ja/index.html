g0vって何？

g0vは、市民の社会への貢献を深め、市民同士をつなぐことを目的とした草の根の市民コミュニティです。 g0vコミュニティを通じて、志を同じくするパートナーを見つけ、草の根的にアイデアを実践し、その結果をオープンライセンス形式で公開することで、より多くの人がその結果に立脚してレースを走ることができるようになります。

govを「0」に置き換え、政府の役割をゼロから見直すg0vは、0と1の世界から来たデジタルネイティブ世代のビジョンでもあるということを表現しています。

運営スタイル

g0v.twは主にオンラインで運営されており、2ヶ月に一度、大規模なハッカソンを開催しています。参加者はさまざまなプロジェクトを提案し、有志を募ります。しばしばハッカソンでのミーティングに加えて、プロジェクトが分離し、大きなg0vイベントとは独立して小さなミーティングやハッカソンを開催することもあります。このような運営方法は、ボトムアップのアイデアを促進し、スラックティビズムよりもアクティビズムを促進します。

（注：ハッカソンとは、Hacker（ハッカー）とMarathon（マラソン）を組み合わせた造語で、1～2日かけて行われる大規模なイベントのこと。有志がグループを作り、作業を行い、その経過を報告します。多くのプロジェクトでは、限られた時間の中でできるだけ多くのことをやり遂げるために、事前にミーティングを行い、当日の計画を立てます。）

コミュニティの構成

g0vのメンバーは、オンラインとオフラインの両方で、あらゆる分野の人が集まっています。唯一の条件は、オープンソース・コラボレーションの原則を信じていることで、ソフトウェア・エンジニアである必要はありません。初期の頃は、g0vのメンバーのほとんどがソフトウェア・エンジニアでしたが、現在では60％以上の参加者が技術的なバックグラウンドを持たない人たちです。

を超えるハッカソンで、多くのプロジェクトが生まれています。全てのプロジェクトではコードもオープンソースにし、オープンソースライセンスで、全ての人に開放しています。以下ではプロジェクトのいくつかを紹介します。

Hackfoldr 黑客公事包

Hackfoldr 為一協作式文件整理介面，讓整理線上文件更容易。

目前已有相當多運用 Hackfoldr 的案例，包括八仙塵爆、高雄氣爆、香港雨傘運動、太陽花學運等社會事件與運動，同時亦為 g0v 大小黑客松最常運用的線上工具之一。

「真的假的」Line Bot

用聊天機器人和群眾協作查核澄清 Line 封閉群組中的不實資訊。

「真的假的 LINE Bot」讓不擅於使用 Google，資訊搜尋能力較低的使用者，可以在加入「真的假的 LINE Bot」之後，遇上任何 LINE 上不知真假的訊息時，直接轉發給機器人，接著就會收到回覆，判斷訊息的真假。

遺落在世界角落的台灣國家寶藏

包括英美等國家這一百多年以來對台灣社會、地理、人物所做下的情蒐紀錄，提供了台灣人在回顧自己的過去時，更豐富的材料與角度，以及不同的思維，去重新描繪台灣人曾經的生活面貌。 在這裡，我們希望將這些資料重新進入公眾視野內，結合每個人獨特的視角，去重新書寫出台灣人生活的故事。

中央政府總預算視覺化

中央政府總預算視覺化專案利用資料視覺化與社群網路的力量提升公民對政府預算規畫的監督能力。

此專案利用各種互動式的視覺化技巧，包含樹狀圖 ( Treemap ) 、泡泡圖 ( Bubble Chart ) 搭配民眾反饋機制來展示預算數字的各種面向，例如歷史趨勢、跨部門比較、公眾意見與稅金的拆解。

プロジェクトハッカソンとローカルコミュニティハッカソン

g0vハッカソンとは別に、g0v-jothonは、基本的なコミュニティインフラストラクチャの維持を支援するために、定期的に「インフラストラクチャハッカソン」を開催しています。

ローカルコミュニティハッカソンはそれぞれ性質が異なっており。多くのイベントで必要としている人材がわかるようになっています。この情報はg0v-facebookグループまたはg0v-slackにアクセスして見つけることができます。

オンライン: g0v facebook と slack のチャンネル

オフラインのイベントに加えて、コミュニティの参加者は、オンラインのディスカッショングループでプロジェクトに関するアイデアを頻繁かつ活発に交換しており、コミュニティからのフィードバックを即座に得られる便利な方法となっています。 現在、主なディスカッショングループは以下の通りです。

  • g0v facebook グループ
  • g0v Slack チャットプラットフォーム
